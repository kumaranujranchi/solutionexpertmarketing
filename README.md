# Expert Solution Marketing (ESM)

This is the official website for Expert Solution Marketing, a digital marketing agency based in Ranchi, Jharkhand.

## Features
- Responsive design using Bootstrap 5
- Dynamic carousel with Owl Carousel
- Contact form
- SEO optimized

## Deployment to Netlify

### Method 1: Continuous Deployment (Recommended)
1. Push this code to your GitHub repository: `kumaranujranchi/solutionexpertmarketing`.
2. Login to [Netlify](https://app.netlify.com/).
3. Click "Add new site" > "Import an existing project".
4. Select "GitHub" and authorize.
5. Search for and select the `solutionexpertmarketing` repository.
6. Leave "Build command" empty.
7. Set "Publish directory" to `.` (or leave it empty as the root is the publish directory).
8. Click "Deploy site".

### Method 2: Manual Drag & Drop
1. Zip the files in the root directory (ensure `index.html` is in the root of the zip).
2. Drag and drop the zip into the Netlify "Sites" dashboard.

## Local Development
To run this project locally:
1. Ensure you have Node.js installed.
2. Run `npx http-server .` in the project directory.
3. Open `http://localhost:8080` in your browser.
